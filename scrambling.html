Scrambling

Scrambling is pretty straightforward, but there are a few things you should know. This guide won't cover notation (you should look up any notation you aren't familiar with, such as clock or Skewb), but it will cover some important information for scrambling.

Scrambling Orientation

For all cubical puzzles, the top face should be the lightest color, and the front should be the darkest possible color. For a normal color scheme, that means white on top and green in front. For Pyraminx, the darkest color should be in front and the lightest on the bottom. For a normal color scheme, that means green in front and yellow on the bottom. For clock, the orientation doesn't matter. For Square-1, the darkest possible scrambling side should be in front.

The Scrambling

When you get a scorecard with a cube on it, check to see how many solves the competitor has done and find the next scramble. For example, if the competitor has done 3 solves, they need scramble 4. Orient the cube properly, and execute the scramble. Then, put both the scorecard and the cube into a cube cover box, and place it near the edge of the table. If a scorecard has a note about an extra scramble, scramble as normal and ignore the note until all the normal scrambles have been given. Then, after the competitor has finished all normal scrambles, they get the extra scramble. If the scorecard has a note about group B (or any other group), make sure the scrambles have been switched to group B before scrambling. Otherwise, scramble a different cube until that happens. If you're scrambling clock, you should consult your delegate about how to transfer clocks to the judging stations, since cube cases can misalign the pins.

You should not scramble for in event in which you haven't competed unless you've discussed the situation with the delegate. If you are scrambling, thanks a lot! It's a huge help for the competition.
